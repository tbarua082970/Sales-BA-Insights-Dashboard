# Sales & Profitability Business Insights Dashboard (Power BI)

## Project Overview
This project demonstrates a mid-level Business Analyst approach to analyzing sales performance, profitability drivers, and customer and regional insights using Power BI.

The dashboard answers key business questions related to revenue growth, margin performance, customer contribution, and regional concentration.

---

## What Was Done in This Project

### Data Preparation & Modeling
- Loaded transactional and master data from CSV and Excel sources
- Cleaned and transformed data using Power Query
- Built a star-schema model with Fact and Dimension tables
- Configured relationships and a proper Date table

### Business Metrics & DAX
- Total Sales
- Total Profit
- Profit Margin %
- Year-over-Year (YoY) analysis
- Year-to-Date (YTD) analysis
- Month-over-Month (MoM) trends

### Dashboard Pages
- **Executive Overview**
- **Profitability & Variance**
- **Customer & Region Insights**

---

## Tools & Technologies
